CMS o Bootstrap?

Para contestar esta pregunta hemos analizado las siguientes opciones que hemos estudiado e utilizado a lo largo de la carrera de Lic. en Tecnologías e Información. Existen muchisimos CMS que son de gran ayuda cuando quieres empezar un proyecto rapidamente y no quieres preocuparte por escribir codigo. La otra opcion es Bootstrap, un framework basado en CSS que funciona solo insertando una pocas lineas de codigo a tus archivos HTML, que proporciona un diseño eficiente, fluido y moderno.

A continuación, vamos analizar las 3 opciones que consideramos para este proyecto:

Herramienta Caracteristicas Ventajas Desventajas
Drupal Es un flexible CMS para desarrollar sitios web que puede ser dirigido a cualquire tipo de negocio, sin embargo no es tan amigable como Wordpress o Joomla. Debido a que es una plataforma nueva y robusta casi no es la primer opcion para desarrolladores de sitios web. Incluye muchas funcionalidades desde el paquete inicial y no requieres instalar demasiados plugins. Instalacion requiere un poco mas de conocimientos tecnicos. Es un CMS moderno y toma un poco mas de tiempo en adaptarse. No esta preparado para la gesition de grandes datos
Wordpress Wordpress inicialmente fue disenada para ser una plataforma de blogs, sin embargo ahora se puede crear hasta paginas complejas y aplicaciones para moviles. Es muy facil y contiene un sistema conveniente para desarrollo web. Es un softare de codigo abierto, preferido por Google, por ende tu sutio tendra una vista previlegiada en su motor de busqueda. Fui desarrollada con los estandares mas altos de calidad de codigo, en cuanto a seguridad es de los mejores y contiene miles de plugins para agregar a tu sitio web. No es un servicio gratuito Requiere de un nivel de conocimientos medio No es la opcion idela para un sitio de ventas Flexibilidad de diseno limitada
Bootstrap Framework de CSS que ofrece un desarrollo responsivo y adaptable a cualquier dispositivo. Funciona mediante archivos de html, css y javascript. Boostrap cuenta con una comunidad enorme que comparte secciones de codigo para usar en tu sitio, y la libreria es inmensa. - Obtienes practica en escribir codigo usanod archivos javascript, html, css, python, etc.
- Es un codigo abierto sin restricciones de uso.
- Te puede ayudar a conseguir un trabajo, desde su lanzamiento hay mas empresas buscando gente con alta capacidad en escribir codigo con bootstrap		 - Requiere de una curva de aprendizaje quisa mas extensa que aprender como funciona un CMS.
